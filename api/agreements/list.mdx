---
title: 'List Commercial Agreements'
api: 'GET /api/acuerdos'
---

Retrieves a list of all commercial agreements with client and salesperson information.

## Query Parameters

<ParamField query="cliente_id" type="integer" optional>
  Filter agreements by client ID
</ParamField>

## Response

Returns an array of commercial agreement objects with joined client and user data.

<ResponseField name="id" type="integer">
  Unique identifier for the agreement
</ResponseField>

<ResponseField name="cliente_id" type="integer">
  Client ID
</ResponseField>

<ResponseField name="cliente_nombre" type="string">
  Client legal name
</ResponseField>

<ResponseField name="cliente_final" type="string">
  End client name
</ResponseField>

<ResponseField name="tarifario" type="string">
  Rate schedule identifier
</ResponseField>

<ResponseField name="desconsolidado_wm" type="string">
  Deconsolidation weight/measurement type
</ResponseField>

<ResponseField name="desconsolidado_min" type="number">
  Minimum deconsolidation rate
</ResponseField>

<ResponseField name="desconsolidado_max" type="number">
  Maximum deconsolidation rate
</ResponseField>

<ResponseField name="observ_desco" type="string">
  Deconsolidation notes
</ResponseField>

<ResponseField name="agp_tn" type="number">
  AGP per ton rate
</ResponseField>

<ResponseField name="manejo_documentos" type="number">
  Document handling fee
</ResponseField>

<ResponseField name="log_fee" type="number">
  Logistics fee
</ResponseField>

<ResponseField name="doc_fee" type="number">
  Documentation fee
</ResponseField>

<ResponseField name="certflete" type="number">
  Freight certificate fee
</ResponseField>

<ResponseField name="seguro_porcentaje" type="number">
  Insurance percentage
</ResponseField>

<ResponseField name="seguro_min" type="number">
  Minimum insurance amount
</ResponseField>

<ResponseField name="emision_bl" type="number">
  Bill of lading emission fee
</ResponseField>

<ResponseField name="plazo_pago" type="string">
  Payment terms
</ResponseField>

<ResponseField name="maximo_descontar" type="number">
  Maximum discount allowed
</ResponseField>

<ResponseField name="comision" type="number">
  Commission rate
</ResponseField>

<ResponseField name="incentivo" type="number">
  Incentive amount
</ResponseField>

<ResponseField name="datos_cotizar" type="string">
  Quote data
</ResponseField>

<ResponseField name="observaciones" type="string">
  General notes
</ResponseField>

<ResponseField name="vendedor_id" type="integer">
  Salesperson user ID
</ResponseField>

<ResponseField name="vendedor_nombre" type="string">
  Salesperson full name
</ResponseField>

<ResponseField name="ultima_mod" type="string">
  Last modification date
</ResponseField>

<ResponseField name="tarifario_fiscal" type="string">
  Fiscal rate schedule
</ResponseField>

<ResponseField name="moneda" type="string">
  Currency (USD, EUR, or ARS)
</ResponseField>

<ResponseField name="created_at" type="string">
  Creation timestamp
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET https://api.iclcotizaciones.com/api/acuerdos \
  -H "Cookie: session=your_session_token"
```

```bash Filter by client
curl -X GET "https://api.iclcotizaciones.com/api/acuerdos?cliente_id=5" \
  -H "Cookie: session=your_session_token"
```

</RequestExample>
