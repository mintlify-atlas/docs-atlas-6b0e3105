---
title: List Quotations
api: GET /api/cotizaciones
---

Retrieves a list of quotations with optional filters.

## Authorization

Requires active session. Non-admin users can only see their own quotations (automatically filtered by `vendedor_id = session.userId`).

## Query Parameters

<ParamField query="fecha_desde" type="string" format="YYYY-MM-DD">
  Filter quotations from this date (inclusive)
</ParamField>

<ParamField query="fecha_hasta" type="string" format="YYYY-MM-DD">
  Filter quotations until this date (inclusive)
</ParamField>

<ParamField query="cliente_id" type="string">
  Comma-separated client IDs. Example: `1,2,5`
</ParamField>

<ParamField query="cerrada" type="string">
  Comma-separated status values. Possible values:
  - `ENCOTIZACION` - In quotation
  - `SI` - Won
  - `NO` - Lost
  - `PEND` - Pending
  - `PEND A/C` - Pending approval
  
  Example: `SI,PEND`
</ParamField>

<ParamField query="vendedor_id" type="string">
  Comma-separated seller IDs. Example: `3,7`
  
  **Note:** Only available for admin users. Non-admin users automatically see only their own quotations.
</ParamField>

<ParamField query="via_id" type="number">
  Filter by via (route) ID
</ParamField>

  </Expandable>

<RequestExample>
```bash cURL
curl -X GET 'https://your-domain.com/api/cotizaciones?fecha_desde=2026-02-01&fecha_hasta=2026-02-28&cerrada=SI,PEND' \
  -H 'Cookie: session=your-session-token'
```

```bash Filter by client
curl -X GET 'https://your-domain.com/api/cotizaciones?cliente_id=3,5' \
  -H 'Cookie: session=your-session-token'
```

```bash Admin: Filter by seller
curl -X GET 'https://your-domain.com/api/cotizaciones?vendedor_id=7' \
  -H 'Cookie: session=your-session-token'
```
</RequestExample>

## Notes

- **Role-based access:** Admin users see all quotations. Non-admin users only see quotations they created.
- Results are ordered by date (descending) and ID (descending)
- All query parameters are optional and can be combined
- Multiple IDs can be passed as comma-separated values for `cliente_id` and `vendedor_id`
