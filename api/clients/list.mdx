---
title: "List Clients"
api: "GET /api/clientes"
description: "Retrieve all clients with assigned salesperson information"
---

Lists all clients in the system with their associated salesperson details.

## Authentication

Requires an active session. Available to all authenticated roles.

## Response

Returns an array of client objects with joined user information.

<ResponseField name="id" type="integer">
  Unique client identifier
</ResponseField>

<ResponseField name="legal_name" type="string">
  Client's legal business name (razón social)
</ResponseField>

<ResponseField name="tax_id" type="string | null">
  Tax identification number (CUIT/CUIL) for invoicing
</ResponseField>

<ResponseField name="phone" type="string | null">
  Contact phone number
</ResponseField>

<ResponseField name="email" type="string | null">
  Contact email address
</ResponseField>

<ResponseField name="client_type" type="enum">
  Type of client business
  
  **Possible values:** `"FFWW"`, `"Final"`, `"Both"`
</ResponseField>

<ResponseField name="user_id" type="integer | null">
  ID of the assigned salesperson (references users table)
</ResponseField>

<ResponseField name="user_name" type="string | null">
  Full name of the assigned salesperson
</ResponseField>

<ResponseField name="is_active" type="boolean">
  Whether the client is currently active
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET https://your-domain.com/api/clientes \
  -H "Cookie: session=your-session-token"
```
</RequestExample>
